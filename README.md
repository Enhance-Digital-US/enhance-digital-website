# Enhance Digital // AI Agency

A modern landing page for an AI-first digital studio offering AI transformation and automation services.

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## Deploy to Netlify

1. Run the build:
   `npm run build`
2. Deploy the dist folder to Netlify:
   `netlify deploy --dir=dist --prod`
